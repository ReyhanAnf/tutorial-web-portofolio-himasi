Hari 2: Membuat Hero Section yang Menarik

Hari ini kita akan fokus membuat bagian pertama yang dilihat pengunjung, yaitu "Hero Section". Bagian ini harus bisa langsung menarik perhatian dengan perkenalan singkat, foto, dan efek ketikan yang dinamis.

Langkah 1: Struktur HTML untuk Hero Section

Tambahkan kode berikut di dalam tag <body>, tepat setelah tag <nav>. Ini akan menjadi wadah utama untuk konten hero section kita.

<!-- Hero Section -->
<section id="home" class="min-h-screen flex items-center bg-gray-100/50 pt-24 md:pt-0">
  <div class="container mx-auto flex flex-col-reverse md:flex-row items-center justify-between gap-10 px-6">
    <!-- Teks Nama -->
    <div class="md:w-1/2 text-center md:text-left">
      <h2 class="text-4xl md:text-6xl font-extrabold mb-4 leading-tight text-gray-900">
        Halo, Saya <br class="hidden md:block"/> <span class="text-teal-600">Ahmad Zaelani</span>
      </h2>
      <p class="text-lg md:text-2xl mb-8 font-medium text-gray-700">
        <span id="typing-effect"></span>
      </p>
      <a href="#projects" class="inline-block bg-teal-600 text-white px-8 py-3 rounded-full font-bold shadow-lg hover:bg-teal-700 transform transition-transform duration-300">
        Lihat Portofolio Saya
      </a>
    </div>
    <!-- Gambar Profil -->
    <div class="md:w-1/2 flex justify-center">
      <div class="relative w-72 h-72 md:w-96 md:h-96">
        <div class="absolute inset-0 bg-teal-200 rounded-full blur-2xl opacity-50"></div>
        <!-- Ubah link nya menjadi gambar kalian atau ubah hurufnya menjadi yang lain -->
        <img src="https://placehold.co/400x400/FFFFFF/00b894?text=RN" alt="Foto Profil" class="relative rounded-full shadow-2xl border-8 border-white w-full h-full object-cover"/>
      </div>
    </div>
  </div>
</section>

Penjelasan Kelas Tailwind CSS:

Kelas Tailwind Fungsi / Deskripsi
min-h-screen flex items-center Membuat section memenuhi tinggi layar dan menengahkan konten secara vertikal.
pt-24 md:pt-0 Memberi padding atas di mobile agar konten tidak tertutup navbar, dan menghapusnya di desktop.
flex-col-reverse md:flex-row Di mobile, gambar akan di atas teks. Di desktop, teks akan di kiri dan gambar di kanan.
font-extrabold, leading-tight Memberi penekanan visual pada judul utama.
relative & absolute Digunakan pada gambar untuk menciptakan efek latar belakang blur yang estetik.
rounded-full shadow-2xl border-8 Membuat gambar menjadi bulat sempurna dengan bayangan tebal dan bingkai putih.

Langkah 2: Menambahkan Efek Ketikan dengan JavaScript

Agar lebih hidup, kita akan menambahkan efek seolah-olah teks sedang diketik. Tambahkan kode JavaScript ini di dalam tag <script> di bagian bawah file HTML Anda, simpan dibawah kode JavaScript pertemuan pertama.


window.addEventListener("scroll", () => {
// ISI PERTEMUAN Pertama
});

// DISINI TEMPAT UNTUK KODE SCRIPT PERTEMUAN 2


// -- Efek Mengetik --
const typingElement = document.getElementById('typing-effect');
const roles = ["Web & Mobile Developer", "UI/UX Enthusiast", "Tech Explorer"];
let roleIndex = 0;
let charIndex = 0;
let isDeleting = false;

function type() {
  const currentRole = roles[roleIndex];
  let displayText = '';

  if (isDeleting) {
    displayText = currentRole.substring(0, charIndex - 1);
    charIndex--;
  } else {
    displayText = currentRole.substring(0, charIndex + 1);
    charIndex++;
  }

  typingElement.textContent = displayText;
  typingElement.style.borderRight = '3px solid #0d9488'; // teal-600

  let typeSpeed = 200;
  if (isDeleting) typeSpeed /= 2;

  if (!isDeleting && charIndex === currentRole.length) {
    typeSpeed = 2000;
    isDeleting = true;
  } else if (isDeleting && charIndex === 0) {
    isDeleting = false;
    roleIndex = (roleIndex + 1) % roles.length;
    typeSpeed = 500;
  }
  
  setTimeout(type, typeSpeed);
}

// Panggil fungsi type saat halaman selesai dimuat
document.addEventListener('DOMContentLoaded', () => {
  if (typingElement) {
    type();
  }
});

Setelah menambahkan kedua bagian kode di atas, simpan file Anda dan buka di browser. Sekarang Anda seharusnya melihat hero section yang menarik dengan efek ketikan yang dinamis!
