Kelas Reguler

Membangun web portofolio dengan Tailwind CSS dan Github Pages serta Implementasi API

Selamat Datang di Kelas Reguler!

Dalam tutorial ini, kita akan bersama-sama membangun sebuah website portofolio dinamis. Kita akan belajar mulai dari dasar HTML, styling dengan Tailwind CSS, hingga mengambil data dari API untuk menampilkan proyek secara dinamis. Tutorial ini dibagi menjadi 6 hari agar mudah diikuti.

Materi Pembelajaran

Hari 1

Struktur & Navbar

Mempersiapkan file, struktur dasar HTML, dan membuat navigasi website yang responsif.

Hari 2

Hero Section

Mendesain bagian utama yang menarik perhatian pengunjung dengan efek ketikan.

Hari 3

Tentang Saya

Membuat bagian untuk memperkenalkan diri dan menampilkan keahlian yang dimiliki.

Hari 4

Formulir Kontak

Membangun formulir kontak yang fungsional dan modern agar pengunjung dapat menghubungi Anda.

Hari 5

Proyek Dinamis (API)

Menampilkan daftar proyek secara dinamis dengan mengambil data dari sebuah API menggunakan JavaScript.

Hari 6

Footer & Finalisasi

Menyelesaikan website dengan footer dan menambahkan interaktivitas akhir dengan JavaScript.
