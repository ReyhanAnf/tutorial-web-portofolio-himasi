Hari 5: Menampilkan Proyek dari API

Selamat datang di materi API! Hari ini kita akan mengubah bagian proyek kita dari statis menjadi dinamis. Kita akan mengambil data dari API publik dan menampilkannya sebagai kartu-kartu proyek yang menarik.

Langkah 1: Struktur HTML untuk Bagian Proyek

Pertama, kita siapkan "wadah" untuk kartu proyek kita. Tambahkan kode ini setelah section kontak. Kita akan mengisi div dengan id="projects-container" menggunakan JavaScript.

<!-- Proyek Section -->
<section id="projects" class="py-24 bg-white">
  <div class="container mx-auto px-6">
    <h3 class="text-3xl font-bold text-center mb-16">Proyek Terbaru</h3>
    <div id="projects-container" class="grid md:grid-cols-2 lg:grid-cols-3 gap-10">
      <!-- Kartu proyek akan dimuat di sini oleh JavaScript -->
    </div>
  </div>
</section>

Kita menggunakan layout grid lagi agar tampilan kartu rapi dan responsif di berbagai ukuran layar.

Langkah 2: Mengambil Data dengan JavaScript Fetch API

Sekarang bagian yang paling seru. Kita akan menggunakan fetch untuk mengambil data dari API. Untuk contoh ini, kita akan menggunakan JSONPlaceholder yang menyediakan data dummy. Tambahkan skrip ini ke file Anda.

async function fetchProjects() {
  const projectsContainer = document.getElementById('projects-container');
  if (!projectsContainer) return;

  try {
    // Menampilkan pesan loading
    projectsContainer.innerHTML = '<p class="col-span-3 text-center text-gray-500">Memuat proyek...</p>';
    
    const response = await fetch('https://jsonplaceholder.typicode.com/posts?_limit=6');
    const projects = await response.json();

    // Membersihkan pesan loading
    projectsContainer.innerHTML = '';

    projects.forEach(project => {
      const projectCard = `
        <div class="bg-white/80 backdrop-blur-md rounded-2xl shadow-lg overflow-hidden group transform hover:-translate-y-2 transition-transform duration-300">
          <div class="relative">
            <img src="https://placehold.co/400x250/333/FFF?text=Proyek" alt="Gambar Proyek" class="w-full h-52 object-cover"/>
            <div class="absolute inset-0 bg-black bg-opacity-40 flex items-center justify-center opacity-0 group-hover:opacity-100 transition-opacity duration-300">
                <a href="#" class="text-white bg-teal-600 px-4 py-2 rounded-full font-semibold">Lihat Detail</a>
            </div>
          </div>
          <div class="p-6">
            <h4 class="text-xl font-bold mb-2 truncate">${project.title}</h4>
            <p class="text-gray-600 mb-4 h-24 overflow-hidden">${project.body}</p>
            <div class="flex flex-wrap gap-2">
              <span class="text-sm bg-blue-100 text-blue-700 px-3 py-1 rounded-full">API</span>
              <span class="text-sm bg-green-100 text-green-700 px-3 py-1 rounded-full">JavaScript</span>
            </div>
          </div>
        </div>
      `;
      projectsContainer.innerHTML += projectCard;
    });

  } catch (error) {
    projectsContainer.innerHTML = '<p class="col-span-3 text-center text-red-500">Gagal memuat proyek.</p>';
    console.error('Error fetching projects:', error);
  }
}

// Panggil fungsi ini saat DOM siap
document.addEventListener('DOMContentLoaded', fetchProjects);

Penjelasan Kode JavaScript:

  • async function fetchProjects(): Kita membuat fungsi asynchronous agar bisa menggunakan await, yang membuat kode lebih mudah dibaca.
  • try...catch: Blok ini digunakan untuk menangani error. Jika API gagal diakses, pesan error akan ditampilkan.
  • await fetch(...): Mengirim permintaan ke URL API dan menunggu respons.
  • await response.json(): Mengubah respons dari format JSON menjadi objek JavaScript.
  • projects.forEach(...): Melakukan iterasi (looping) untuk setiap item proyek yang didapat dari API.
  • Template Literal (``): Kita membuat string HTML untuk kartu proyek. Ini cara modern untuk menggabungkan string dan variabel (seperti ${project.title}).
  • projectsContainer.innerHTML += ...: Menambahkan HTML kartu proyek yang baru dibuat ke dalam div container.
