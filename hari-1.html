Hari 1: Struktur Dasar & Navbar

Selamat datang di hari pertama! Hari ini kita akan menyiapkan fondasi website kita. Kita akan membuat struktur dasar file HTML dan membangun sebuah navbar yang keren dan responsif.

Langkah 1: Struktur Dasar HTML

Pertama, buat file baru bernama index.html. Kemudian, salin dan tempelkan kode di bawah ini. Ini adalah "kerangka" dari setiap halaman web.

<!DOCTYPE html>
<html lang="id" class="scroll-smooth">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Portfolio Interaktif</title>
    <script src="https://cdn.tailwindcss.com"></script>
    <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap" rel="stylesheet" />
    <script src="https://unpkg.com/lucide@latest"></script>
    <style> body { font-family: 'Inter', sans-serif; } </style>
  </head>
  <body class="bg-gray-50 text-gray-800">
    <!-- Konten website kita akan ada di sini -->
  </body>
</html>

Penjelasan Kode:

  • <!DOCTYPE html>: Memberi tahu browser bahwa ini adalah dokumen HTML5.
  • <html lang="id">: Tag pembuka utama, menandakan bahasa konten adalah Bahasa Indonesia.
  • <head>: Berisi informasi meta tentang halaman (judul, skrip, stylesheet).
  • <meta name="viewport" ...>: Ini sangat penting untuk desain responsif. Halaman akan menyesuaikan lebarnya dengan lebar layar perangkat.
  • Script & Link Tags: Kita memuat Tailwind CSS, Google Fonts, dan Lucide Icons.
  • <body>: Di sinilah semua konten yang terlihat oleh pengguna akan ditempatkan.

Langkah 2: Membuat Navbar

Navbar adalah menu navigasi di bagian atas website. Kita akan membuatnya tetap di atas saat halaman di-scroll dan responsif untuk tampilan mobile.

<!-- Navbar -->
<nav id="navbar" class="fixed w-full top-0 bg-white/80 backdrop-blur-md shadow-sm z-50">
  <div class="container mx-auto flex justify-between items-center p-4">
    <a href="#home" class="text-2xl font-bold text-teal-600">NamaAnda</a>
    <ul id="desktop-menu" class="hidden md:flex space-x-8">
      <li><a href="#home" class="hover:text-teal-600">Home</a></li>
      <li><a href="#about" class="hover:text-teal-600">Tentang</a></li>
      <li><a href="#projects" class="hover:text-teal-600">Proyek</a></li>
      <li><a href="#contact" class="hover:text-teal-600">Kontak</a></li>
    </ul>
    <button id="hamburger-btn" class="md:hidden z-50">
      <i data-lucide="menu" class="w-7 h-7"></i>
    </button>
  </div>

  <!-- Menu untuk Mobile -->
  <div id="mobile-menu" class="hidden md:hidden absolute top-0 left-0 w-full h-screen bg-white">
    <ul class="flex flex-col items-center justify-center h-full space-y-8 text-xl font-medium">
        <li><a href="#home" class="nav-link-mobile">Home</a></li>
        <li><a href="#about" class="nav-link-mobile">Tentang</a></li>
        <li><a href="#projects" class="nav-link-mobile">Proyek</a></li>
        <li><a href="#contact" class="nav-link-mobile">Kontak</a></li>
    </ul>
  </div>
</nav>

Langkah 3: Membuat Interaksi Menggunakan JavaScript

JavaScript adalah bahasa pemrograman yang memungkinkan kita membuat halaman web dan aplikasi menjadi interaktif dan dinamis, bukan hanya menampilkan konten statis

Tambahkan Tag Script Diatas penutup tag body


  <script>
    // <!-- Tempat menyimpan kode JavaScript -->
  </script>

  <!-- Nah posisinya di atas ini -->
  </body> <!-- Yang Ini jangan di copy -->

Tambahkan Kode untuk responsif Navbar


  // -- Navbar & Menu Mobile --
  const navbar = document.getElementById('navbar');
  const hamburgerBtn = document.getElementById('hamburger-btn');
  const mobileMenu = document.getElementById('mobile-menu');
  const navLinks = document.querySelectorAll('.nav-link');
  const mobileNavLinks = document.querySelectorAll('.nav-link-mobile');
  const sections = document.querySelectorAll('section');

  // Toggle Menu Mobile
  hamburgerBtn.addEventListener('click', () => {
    const isHidden = mobileMenu.classList.contains('hidden');
    if(isHidden) {
      mobileMenu.classList.remove('hidden');
      hamburgerBtn.innerHTML = '';
    } else {
      mobileMenu.classList.add('hidden');
      hamburgerBtn.innerHTML = '';
    }
    lucide.createIcons(); // Re-render icon
  });

  // Tutup menu saat link di-klik (untuk mobile)
  mobileNavLinks.forEach(link => {
    link.addEventListener('click', () => {
      mobileMenu.classList.add('hidden');
      hamburgerBtn.innerHTML = '';
      lucide.createIcons();
    });
  });

  // Efek shadow pada navbar saat scroll
  window.addEventListener('scroll', () => {
    if (window.scrollY > 50) {
      navbar.classList.add('shadow-md');
      navbar.classList.remove('shadow-sm');
    } else {
      navbar.classList.remove('shadow-md');
      navbar.classList.add('shadow-sm');
    }

    // Active link highlighting on scroll
    let current = '';
    sections.forEach(section => {
      const sectionTop = section.offsetTop;
      if (pageYOffset >= sectionTop - 100) {
        current = section.getAttribute('id');
      }
    });

    navLinks.forEach(link => {
      link.classList.remove('active');
      if (link.getAttribute('href').substring(1) === current) {
        link.classList.add('active');
      }
    });
  });

Penjelasan Kelas Tailwind CSS di Navbar:

  • fixed top-0 w-full z-50: Membuat navbar menempel di bagian atas layar.
  • bg-white/80 backdrop-blur-md: Memberi efek "glassmorphism" transparan dan buram.
  • container mx-auto: Menengahkan konten di dalam navbar.
  • flex justify-between items-center: Mengatur layout item navbar menggunakan Flexbox.
  • hidden md:flex: Menyembunyikan menu desktop di layar kecil dan menampilkannya di layar medium ke atas.
  • md:hidden: Menampilkan tombol hamburger hanya di layar kecil.
