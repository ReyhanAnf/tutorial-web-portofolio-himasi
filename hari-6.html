Hari 6: Sentuhan Akhir dengan Footer & JavaScript

Selamat datang di hari terakhir! Kita akan menyelesaikan website kita dengan menambahkan footer, ikon sosial media, dan menggabungkan semua fungsi JavaScript yang diperlukan untuk interaktivitas.

Langkah 1: Membuat Footer

Footer adalah bagian paling bawah dari website, biasanya berisi info hak cipta dan link sosial media. Tambahkan kode ini sebelum tag penutup </body>.

<!-- Footer -->
<footer class="bg-gray-800 text-white">
  <div class="container mx-auto py-10 px-6 text-center">
    <div class="flex justify-center space-x-6 mb-6">
      <a href="#" class="hover:text-teal-400 transition"><i data-lucide="github" class="w-7 h-7"></i></a>
      <a href="#" class="hover:text-teal-400 transition"><i data-lucide="linkedin" class="w-7 h-7"></i></a>
      <a href="#" class="hover:text-teal-400 transition"><i data-lucide="instagram" class="w-7 h-7"></i></a>
    </div>
    <p>© 2025 NamaAnda. Dibuat dengan santuy di Kelas Reguler Himasi.</p>
  </div>
</footer>

Penjelasan Kelas Tailwind CSS:

Kelas / Properti Penjelasan
bg-gray-800 text-white Memberi latar belakang berwarna abu gelap (gray-800) dan teks berwarna putih untuk menciptakan kontras yang jelas pada bagian footer.
flex justify-center space-x-6 Menampilkan ikon sosial media secara sejajar menggunakan Flexbox, menengahkan posisinya, serta memberi jarak antar ikon sebesar 1.5rem.
hover:text-teal-400 transition Mengubah warna ikon menjadi teal saat diarahkan oleh kursor dengan efek transisi halus untuk interaksi yang lebih elegan.

Langkah 2: Struktur Akhir Kode JavaScript

Sekarang, pastikan kode JavaScript yang kamu punya strukturnya mirip seperti ini

document.addEventListener('DOMContentLoaded', () => {
    lucide.createIcons();

    // -- Toggle Menu Mobile Navbar --
    const hamburgerBtn = document.getElementById('hamburger-btn');
    const mobileMenu = document.getElementById('mobile-menu');
    const mobileNavLinks = document.querySelectorAll('.nav-link-mobile');

    if(hamburgerBtn) {
        hamburgerBtn.addEventListener('click', () => {
            mobileMenu.classList.toggle('hidden');
            const icon = mobileMenu.classList.contains('hidden') ? 'menu' : 'x';
            hamburgerBtn.innerHTML = `<i data-lucide="${icon}" class="w-7 h-7"></i>`;
            lucide.createIcons();
        });

        mobileNavLinks.forEach(link => {
            link.addEventListener('click', () => {
                mobileMenu.classList.add('hidden');
                hamburgerBtn.innerHTML = `<i data-lucide="menu" class="w-7 h-7"></i>`;
                lucide.createIcons();
            });
        });
    }

    // -- Efek Mengetik --
    const typingElement = document.getElementById('typing-effect');

    }

    async function fetchProjects() {

});

Penjelasan Lengkap JavaScript:

Bagian Kode Penjelasan
document.addEventListener('DOMContentLoaded', ...) Menjalankan seluruh skrip setelah halaman selesai dimuat. Ini mencegah error ketika elemen HTML belum siap.
lucide.createIcons() Mengaktifkan ikon dari library Lucide agar bisa tampil di halaman. Wajib dipanggil ulang setelah manipulasi DOM.
hamburgerBtn.addEventListener('click', ...) Menambahkan event klik pada tombol menu mobile. Ketika diklik, kelas hidden di-toggle untuk membuka/tutup navigasi.
mobileNavLinks.forEach(link => {...}) Menutup menu otomatis saat salah satu link navigasi diklik, menjaga pengalaman pengguna yang rapi di mode mobile.
const typingElement = document.getElementById('typing-effect') Mengambil elemen teks untuk efek mengetik (typing effect). Efek ini berasal dari latihan di Hari 2.
async function fetchProjects() Fungsi asynchronous untuk mengambil data proyek dari API. Menggunakan await agar penulisan kode lebih bersih dan mudah dibaca.
await fetch(...) Mengirim request HTTP ke server. await akan menunggu hingga data diterima sebelum lanjut ke langkah berikutnya.
await response.json() Mengonversi hasil dari API (berformat JSON) menjadi objek JavaScript agar bisa diolah.
projectsContainer.innerHTML = ... Mengubah isi HTML dari container proyek. Awalnya menampilkan teks ‚ÄúMemuat...‚Äù, kemudian diganti dengan kartu proyek dinamis.
forEach(project => ...) Melakukan iterasi pada data proyek dan membuat HTML untuk setiap item menggunakan Template Literal.
Template Literal (``) Memungkinkan penulisan HTML multi-baris dan menyisipkan variabel dengan ${variable}. Cara modern mengganti string concatenation.
try...catch Menangani error jaringan atau server API. Jika gagal, menampilkan pesan error ramah pengguna dan log di konsol.
console.error() Menampilkan detail error di konsol browser untuk debugging tanpa mengganggu pengguna.
